numpadDecSeparator - demo

Numpad decimal separator plugin for jQuery - version
code google jQuery plugin site

Intro

With this jQuery plugin you can configure what character to use for the numpad decimal separator.

The examples below show you how to use the plugin.

Example 1: Comma as numpad decimal separator - Default

Usage example

					$(function(){
					 $("#default").numpadDecSeparator();
					});
					...
					<input type="text" id="default" />

Example 2: Space as numpad decimal separator

Usage example

					$(function(){
					  $("#space").numpadDecSeparator({separator: ' '});
					});
					...
					<input type="text" id="space" />

Example 3: Use of predefined variables

As separator option you can use one of the two default predefined variables (SPACE and COMMA).

SPACE => will output space.

COMMA => will output comma.

Usage example

					$(function(){
					  $("#predefinedVariable").numpadDecSeparator({separator: 'SPACE'});
					  // this is the same as 
					  // $("#predefinedVariable").numpadDecSeparator({separator: ' '});	
					});
					...
					<input type="text" id="predefinedVariable" />

Example 4: Use regional setting of operating system as numpad decimal separator

'150.00' will output '150,00' when regional setting is dutch belgium

'150.00' will output '150.00' when regional setting is english US

This only works in firefox and IE7+!

Usage example

					$(function(){
					 $("#regional").numpadDecSeparator({useRegionalSettings: true});
					});
					...
					<input type="text" id="regional" />

Example 5: Adding fields dynamically

If you use the numpadDecSeparator in a dynamic environment as an ajax environment where you add fields on the fly you can use the following example as a guide.

Usage example

					$(function(){
					 $("#default").numpadDecSeparator();
					});
					function addInput(){
					 $('<input type="text" id="amount3" class="add" />')
					 		.appendTo('#addInput');
					 $(".add").numpadDecSeparator();
					}
					...
					<input type="text" class="add" />
					<input type="button" value="add input" onclick="addInput();" />

Example 6: Unbind

Usage example

					$(function(){
					 $(".unbind").numpadDecSeparator({separator: ' '});
					 $("#unbindButton").click(function(){
					   $(".unbind").numpadDecSeparator('unbind').val("").first().focus();
					 });
					});
					...
					<input type="text" class="unbind" />
					<input type="button" id="unbindButton" value="unbind" />

before unbind typing '150.00' with the numpad will output '150 00', after unbind it will output '150.00'

Example 7: Overriding default options

Usage examples

					//override default separator
					$.fn.numpadDecSeparator('mergeDefaults', {separator: "SPACE"});
					//or
					//override default useRegionalSettings
					$.fn.numpadDecSeparator('mergeDefaults', {useRegionalSettings: true});
					//or
					//override default predefinedVariables
					$.fn.numpadDecSeparator('mergeDefaults', {predefinedVariables: {APOSTROPHE: "'"}});
					//or
					//override all defaults
					var newDefaults = {
						separator : " ",
						useRegionalSettings : true,
						predefinedVariables: {SPACE: " "}
					 };
					 $.fn.numpadDecSeparator('mergeDefaults', newDefaults);
