numpadDecSeparator - demo

Numpad decimal separator plugin for jQuery
home

Example 1: Comma as numpad decimal separator - default

Usage example

					$(function(){
					 $("#default").numpadDecSeparator();
					});
					...
					<input type="text" id="default" />

Example 2: Space as numpad decimal separator

Usage example

					$(function(){
					 $("#space").numpadDecSeparator({separator: ' '});
					});
					...
					<input type="text" id="space" />

Example 3: Use regional setting of operating system as numpad decimal separator

'150.00' will output '150,00' when regional setting is dutch belgium

'150.00' will output '150.00' when regional setting is english US

This only works in firefox and IE7+!

Usage example

					$(function(){
					 $("#regional").numpadDecSeparator({useRegionalSettings: true});
					});
					...
					<input type="text" id="regional" />

Example 4: Adding fields dynamically

If you use the numpadDecSeparator in a dynamic environment as an ajax environment where you add fields on the fly you can use the following example as a guide.

Usage example

					$(function(){
					 $("#default").numpadDecSeparator();
					});
					function addInput(){
					 $('<input type="text" id="amount3" class="add" />')
					 		.appendTo('#addInput');
					 $(".add").numpadDecSeparator();
					}
					...
					<input type="text" class="add" />
					<input type="button" value="add input" onclick="addInput();" />

Example 5: unbind

Usage example

					$(function(){
					 $(".unbind").numpadDecSeparator({separator: ' '});
					 $("#unbindButton").click(function(){
					   $(".unbind").numpadDecSeparator('unbind').val("").first().focus();
					 });
					});
					...
					<input type="text" class="unbind" />
					<input type="button" id="unbindButton" value="unbind" />

before unbind '150.00' will output '150 00', after unbind will output '150.00'
